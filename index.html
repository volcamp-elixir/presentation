### Pattern matching ```elixir x = 1 %{"number" => x} = %{"number" => 1} {_, x} = {42, 1} "x = " <> x = "x = 1" [x | _ ] = [1, 2, 3] 1 = x ```
### Enum and Streams Bye bye, foreach loop ! ```elixir list = [1, 2 , 3] sum = list |> Stream.map(fn x -> x * 2 end) |> Stream.filter(fn x -> x < 5 end) |> Enum.sum() 6 = sum ```
### Async process Bye bye, foreach loop ! ```elixir # run every thnig in detached process spawn(fn -> my_heavy_function() end) # combine stream and async [1,2,3] |> Task.async_stream(fn x -> my_heavy_function end) |> Stream.run() ```
